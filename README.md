## 📖 Sobre o Projeto
 
Inspirado no misterioso enigma **Cicada 3301** — um dos puzzles mais complexos da internet — este projeto recria a experiência de resolver desafios criptográficos em etapas, onde cada fase revela pistas para a próxima.
 
O usuário deve decifrar mensagens ocultas, códigos e enigmas para avançar nas fases. Cada desafio exige um tipo diferente de raciocínio e conhecimento técnico.

## 🛠️ Tecnologias
 
- **Backend:** Python + Flask
- **Frontend:** HTML5 + CSS3
- **Esteganografia:** Script LSB customizado (`encode_steg.py`)
- **Metadados:** EXIF
- **Versionamento:** Git + GitHub

- ## 🚀 Como Rodar Localmente
 
**Pré-requisitos:** Python 3.x instalado
 
```bash
# 1. Clone o repositório
git clone https://github.com/Pedroaruana/Cicada-3301.git
cd Cicada-3301
 
# 2. Instale as dependências
pip install flask
 
# 3. Inicie o servidor
python app.py
 
# 4. Acesse no navegador
http://localhost:5000
```
